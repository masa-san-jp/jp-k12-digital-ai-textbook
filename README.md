# jp-k12-digital-ai-textbook-spec

日本の小学校1年生〜高校3年生を対象とした、日本最高峰の「デジタル・AI人材育成」教科書シリーズを目指す実験的・社会貢献型のオープンソースプロジェクト。

## プロジェクトの位置づけ

- 主催者: masa-san-jp
- 特定顧客への受託仕様書ではなく、教育の質を社会全体に還元することを目的とするOSSプロジェクトである
- Claude Fable 5（Anthropic）を用いて、政策文書・国際フレームワークの調査、仮説設計、構造化、推敲を大規模に行い、人手のみでは到達しづらい網羅性・一貫性・再現性を追求している
- 誰でも自由に閲覧・改変・再配布・実装できることを前提に設計する（詳細はライセンス参照）
- 最終的な採否・方針決定はプロジェクトオーナーが行い、本リポジトリの文書群はその意思決定を支えるための判断材料・仕様案という位置づけである

## 免責事項

- 本リポジトリは特定の事業者による教科書制作プロジェクトのための提案・仕様文書であり、文部科学省・中央教育審議会その他公的機関の公式見解・承認を示すものではない
- 次期学習指導要領は2026年7月時点で審議中であり、本文書中の学年配当・時数・領域名（例:「情報・技術科（仮称）」）は変更される可能性がある。確定情報は文部科学省の公表資料を一次情報として参照すること
- 本文書に記載の目標ID・単元名・成果指標はすべて設計上の仮案であり、実証授業（品質ゲートG3）による検証前の値を含む

## 目的

どの出版社・どの編集者が制作を担っても同一水準の教育品質が再現されるよう、12学年分の教科書・指導書・デジタル教材の仕様を構造化・文書化する。次期学習指導要領（2026年度答申予定）およびUNESCO AI Competency Framework for Students（2024）に整合させ、国内規範と国際水準の両方を満たすことを目標とする。

## 設計原則（要約）

- **人間中心・原理主義・創り手育成・スパイラル構造・公平性**の5原則を全学年で一貫させる
- 到達目標は `[学年]-[領域]-[連番]` のID管理とし、`条件・対象・行動・水準` を含む文で記述する（解釈のブレを排除）
- 全単元に「アンプラグド代替」「倫理討議」「差異化3水準（基礎/標準/発展）」を必置
- 5段階品質ゲート（G1カリキュラム承認〜G5統合整合）を全巻が通過することを制作プロセスとして義務化
- 生成AIの児童生徒による直接利用は学年別に段階規定（小1-4は教師実演のみ／小5-6は管理下限定利用／中1以上は主体利用＋AI利用申告）

## 現状の既知の未完部分（要判断・要作業）

学年別詳細設計書 第7章に自己監査結果を記載済み。次のうち着手順序はオーナー判断が必要。

1. 高2・高3の非代表目標の全文展開（現在は代表目標のみ確定、残りは概要記述）
2. 各単元の差異化3水準（基礎/標準/発展）の具体活動記述
3. 単元別時数の実証授業による実測補正（現状は配分ルールに基づく暫定値）
4. 付録A〜H（トレーサビリティ表・用語辞書・執筆チェックリスト等）の実体制作

## 用語・命名規則

- 目標ID: `[学年コード]-[領域A〜E]-[連番2桁]`（例: `E3-B-02` = 小3・データとAI・2番目の目標）
- ★印: 次学年の前提となるコア目標（実証授業での達成率基準が非★より高い）
- 学年コード: `E1〜E6`＝小1〜6、`J1〜J3`＝中1〜3、`H1〜H3`＝高1〜3
